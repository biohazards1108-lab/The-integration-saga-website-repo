About the Integration Saga

Z-VIRUS OUTBREAK // CLASSIFIED OVERVIEW

The Integration Saga follows survivors trapped in a world reshaped by the Z-Virus: a catastrophic outbreak that mutates humans into terrifying infected forms. Each book pushes deeper into the mystery of PROJECT EDEN, government experiments, and the cost of survival.

This terminal is built as if it were an in-universe system: a classified database cataloging survivors, infected, locations, and events from the series.

If you’re new, start with the books. If you’re already invested, use this terminal to spot easter eggs, track timelines, and explore the world in more detail.
